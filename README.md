# Dev

1. Copiar el ```env.template``` y renombrar a ```.env```

2. Ejecutar 
```
npm install
```

3. Levantar la imagen de (Docker desktop)
```
docker-compose up -d
```

4. Levantar nest 
```
npm run start:dev
```
 
5. Ir a 
```
localhost:3000/graphql
```
